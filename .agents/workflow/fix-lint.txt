# Fix Lint Workflow

1. **PHP:** Check for syntax errors by running `php -l <file>` on modified files.
2. **HTML:** Validate proper tag nesting, missing `alt` attributes on images, and unclosed tags.
3. **CSS:** Look for duplicate selectors, unused rules, and missing semicolons.
4. **JS:** Check for undeclared variables, missing `const`/`let`, and uncaught promise rejections.
5. **General:** Ensure consistent indentation (4 spaces for PHP, 2 spaces for JS/CSS/HTML).
6. **Security Lint:** Flag any raw `$_GET`/`$_POST` usage without sanitization.
